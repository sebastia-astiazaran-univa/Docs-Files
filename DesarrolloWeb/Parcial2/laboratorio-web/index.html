Semana 1

Color que si sirve en web

Trabaja con una paleta limitada y observa como cambia la personalidad de la interfaz.

Paletas sugeridas

  • Usa un color principal para identidad.
  • Usa el acento solo para botones o foco visual.
  • El texto necesita contraste alto.
  • No todo debe competir por llamar la atencion.
