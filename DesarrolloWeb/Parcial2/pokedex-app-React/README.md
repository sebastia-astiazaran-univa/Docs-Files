# Pokédex Nacional (React + TypeScript)

Versión en **React** de la Pokédex vanilla ubicada en `../pokedex-app`, creada con **Vite**.

## Requisitos

- Node.js LTS + npm

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
npm run preview
```
