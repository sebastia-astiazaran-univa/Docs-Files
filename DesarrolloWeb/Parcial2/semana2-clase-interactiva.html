▸ Acordeón (FAQ)

¿Qué es Flexbox? ▼
Flexbox es un sistema de layout unidimensional. Controla cómo los elementos se distribuyen a lo largo de un eje (fila o columna). Perfecto para navbars, filas de botones y alineación vertical.
¿Cuándo usar Grid vs Flexbox? ▼
Usa Grid para layouts de dos dimensiones (columnas Y filas al mismo tiempo). Usa Flexbox cuando solo necesitas controlar un eje. En la práctica, se complementan.
¿Qué es mobile-first? ▼
Diseñar primero para móvil y luego expandir con media queries (min-width). Es más fácil agregar complejidad que quitarla. Además, Google indexa la versión móvil primero.
¿Qué hace border-radius? ▼
Redondea las esquinas de un elemento. Valores pequeños (4–8px) dan estilo profesional sutil. Valores grandes (16–20px) dan aspecto moderno y suave. border-radius: 50% crea círculos.
